# Vector Quantization Coding
Encode and decode a file using the Linde-Buzo-Gray algorithm


### Install

```
    $ pip install -e .
```
  
### Using

Encoding a file:

```
    $ lbg <file> [options]
    $ lbg -t <text> [options]
```

Decoding a file:

```
    $ lbg <file.table | file.acod> -d [options]
```
  
More options:

```
    $ lbg --help
```
